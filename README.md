# 📚 R Dictionary

Repositório criado para reunir funções, comandos e exemplos da linguagem **R**, funcionando como um dicionário de consulta rápida.

O objetivo é organizar os principais pacotes utilizados em Ciência de Dados e Estatística, contendo:

- descrição da função;
- sintaxe;
- argumentos;
- exemplos práticos;
- quando utilizar;
- documentação oficial.
